# ReportRefiner

Google ドキュメント向けの Google Apps Script（GAS）アドオンです。技術レポートをワンクリックで整形し、表記ゆれ吸収・レイアウト調整・図表／見出しの採番などを自動化します。

## 主な機能

| メニュー | 概要 |
| --- | --- |
| スタイル調整（文字・用語） | `Style.js` で全角英数を半角へ変換し、`Master.js` で技術用語や単位を統一します。 |
| レイアウト調整（画像中央） | `Layout.js` の `alignCenterLayout()` が画像段落のインデントを解除して中央揃えにします。 |
| インデント自動調整 | 図表と見出しを除く段落を「左 0 / 初行 1 文字」に統一し、箇条書きも安全に再構築します。 |
| 見出し番号付け | `Heading.js` の `addHeadingNumbers()` が最大 6 階層の階層番号を自動付与します。 |
| 見出し番号削除 | `removeHeadingNumbers()` で付与済みの番号だけを取り除きます。 |
| 図表番号・タイトル追加 | `Figure.js` が見出し番号と連動した図表番号を生成し、キャプションを中央寄せで挿入します。 |

## 導入手順

1. 整形したい Google ドキュメントを開きます。
2. メニューから **拡張機能 → Apps Script** を開き、本リポジトリ内の各ファイル（`Main.js` など）をスクリプトエディタに貼り付けます。
3. `appsscript.json` の内容でプロジェクト設定を置き換え、スクリプトプロパティを保存します。
4. 初回実行時に表示される承認ダイアログで権限を付与します。
5. ドキュメントに戻り、メニューに追加された **「カスタム機能」** から任意の項目を実行します。

## 使い方のヒント

- **処理の推奨順序**：見出し番号 → 図表番号 → インデント／レイアウト → スタイル調整。
- **図表採番の前提**：少なくとも 1 つの見出しに番号が付いていないと `addFigureNumbers()` はエラーで停止します。
- **辞書のカスタマイズ**：
  - 用語統一は `Master.js` の `SIMPLE_REPLACEMENTS` を編集します。
  - 単位とフォーマットは `UNITS` と `REGEX_REPLACEMENTS` を調整します。
  - 全角→半角変換を追加したい場合は `Style.js` の `safeReplace()` 呼び出しを追記してください。
- **インデント調整の制御**：図表キャプションや中央揃え段落は自動的にスキップされます。追加で除外したい段落があれば、判定条件を `adjustIndent()` に加えてください。

## 開発とデプロイ（VS Code + clasp）

1. 事前準備として `npm install -g @google/clasp` で clasp を導入し、`clasp login` で使用する Google アカウントを認証します。
2. 既存プロジェクトを取り込む場合は `clasp clone <scriptId>`、新規の場合は `clasp create --type docs --title "ReportRefiner"` などで GAS プロジェクトを作成します。
3. プロジェクト直下に `Main.js` などのスクリプトと `appsscript.json` を配置し、VS Code で編集します。
4. GAS 側の変更を取得したい場合は `clasp pull`、ローカル変更をアップロードする場合は `clasp push` を実行します。
  - 強制上書きが必要なときは `clasp push -f` を使用します。
  - `clasp push` では `appsscript.json` も同時に更新されるため、設定変更後の反映漏れを防げます。
5. push 後は Google ドキュメントを開き、メニュー「カスタム機能」から対象処理を実行して動作を確認します。

## フォルダー構成

```
Main.js        // メニューとエントリーポイント
Style.js       // 全角→半角変換
Master.js      // 用語・単位の表記統一
Layout.js      // レイアウト（画像中央揃え・インデント調整）
Heading.js     // 見出し番号の付与／削除
Figure.js      // 図表番号・キャプションの生成
appsscript.json// GAS プロジェクト設定
```

## トラブルシューティング

- **メニューが出ない**：`onOpen()` が動かない場合は Apps Script エディタから `onOpen` を一度手動実行してください。
- **権限エラー**：エディタ右上の実行ボタンから任意の処理を実行し、Google アカウント認証を完了させます。
- **採番がずれる**：見出しテキストに手動で付けた番号が残っていると重複するため、先に「見出し番号削除」を実行してから付け直してください。
- **図表がスキップされる**：1x1 の表はコードブロックと見なしてスキップします。通常表として扱いたい場合はセルを増やしてください。
